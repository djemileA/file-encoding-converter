# File Encoding Converter

Kotlin service for automatic file encoding detection and conversion to Windows-1251.

## Features
- Automatic encoding detection (UTF-8, UTF-16, Windows-1251, etc.)
- Console output of file content
- Save file in Windows-1251 encoding
- Supports multiple text encodings

## Supported Encodings
- UTF-8 (with and without BOM)
- UTF-16LE / UTF-16BE
- Windows-1251
- KOI8-R
- ISO-8859-1
- US-ASCII
